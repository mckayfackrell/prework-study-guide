HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Enter your HTML notes header
  • h1,h2, h3,h4,h5 and h6: different headings and subheadings
  • p: paragraph or block of text
  • ul,ol, and li: unordered lists, ordered lists, and list items
      (so the list must begin with ul and then the items must have li before them)
  • img: contains information about an image on the webpage
    • src: has information about the specific image and where it is on the webpage
    • alt: alternative name that describes the image and is useful for screen readers
    • width: ie 520
    • height: ie 346
  • br: creates a line of empty space or a line break
  • a: creates links to same webpage or other webpages
      href attributes points to the URL for the link ie < a href="https://apple.com">Apple Apple

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS is broken into three styles: inline CSS, internal CSS style sheet, and external CSS style sheet. Inline CSS: when a developer wants to style an element directly in the HTML file
  • The best case scenario is to create a style sheet as a separate sheet where the elements live
  • In order to link the stylesheet you must add code after the head and before the title (after the last meta)
      • < link rel="stylesheet" href="./assets/style.css" >
